# 广告费用与产品销量回归分析

## 项目简介

本项目使用多种线性回归模型分析广告投放（电视、广播、报纸）与产品销量之间的关系。通过对比不同模型的性能，为广告投放策略提供数据支撑。

## 功能特性

- 支持4种回归模型：最小二乘法、岭回归、Lasso回归、弹性网络回归
- 自动超参数调优（网格搜索）
- 完整的数据预处理与特征标准化流程
- 多维度模型性能对比（RMSE、R²分数）
- 绘制性能对比图、预测值对比图、特征重要性图
- 数据缺失自动生成真实合成样本

## 安装与使用

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行主程序

```bash
python "1 广告费用与产品销量.py"
```

程序会自动加载数据、训练模型、生成图表。

### 3. 生成或更新数据

如需重新生成合成数据：

```bash
python generate_ad_data.py
```

支持自定义参数（样本数、随机种子）：

```bash
python generate_ad_data.py 150 2026
```

## 输出结果

程序运行后在当前目录生成以下文件：

- `model_performance_comparison.png` - 各模型RMSE和R²分数对比
- `prediction_comparison.png` - 测试集预测值与真实值比较
- `feature_importance.png` - 线性模型特征系数

控制台输出包括：
- 各模型的训练/测试集评估指标

## 数据说明

### 数据格式

数据文件包含4列：
- TV：电视广告投入（单位：千元）
- radio：广播广告投入（单位：千元）
- newspaper：报纸广告投入（单位：千元）
- sales：产品销量（单位：万件）

### 合成数据

若缺少原始数据，程序自动生成逼真的合成样本：
- 样本数：100条
- 特征分布：根据真实广告市场数据模拟
- 包含交互项和异方差噪声，增强真实性

## 模型说明

### 最小二乘法

基础线性回归模型，无正则化项。用作基准对比。

### 岭回归

加入L2正则化，防止过拟合。参数alpha通过5折交叉验证自动选择。

### Lasso回归

加入L1正则化，支持自动特征选择。参数alpha通过5折交叉验证自动选择。

### 弹性网络回归

结合L1和L2正则化，参数alpha和l1_ratio通过网格搜索自动选择。
